# InnoReader 阅读器框架

InnoReader 阅读器框架集成了多个精选开源作品于一身，实现了如可滑动左边菜单栏、滚动展示页、上下拉刷新、可滚动切换页面等等多种功能。用户可根据自己的需求方便的自定义并快速部署。
